Containers with Docker

## Objectives 

1. Install Docker
2. Write a `Dockerfile` and build a Docker image

## 1. Installer Docker

Pour installer Docker on doit utiliser la commande: 
```
docker run hello-world
```
et on recoit en retour ce message: 'Hello from Docker!'
